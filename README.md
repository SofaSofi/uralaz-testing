# Тестирование веб-интерфейса портала ООО «АЗ УРАЛ»

## Описание
Автоматизированное тестирование с использованием Python, pytest и Selenium WebDriver.

## Структура проекта
- `conftest.py` — фикстуры pytest
- `base_page.py` — базовый класс Page Object
- `main_page.py` — главная страница
- `tests/` — тестовые сценарии

## Запуск
pip install -r requirements.txt
pytest -v
